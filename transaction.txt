 **TransactionStatus**: Represents the status of the transaction.
- 'id' (CHAR(1)): Unique identifier for each transaction status. refers to the 'status' field in the transaction table.
- 'trantype' (VARCHAR2): Type of the transaction. refers to the 'type' field in the transaction table.
- 'name' (VARCHAR2): Name of the transaction status. 'Open', 'Closed, 'Voided', etc..
- 'fullname' (VARCHAR2): Description of the transaction status.

                ### Example Query:

                To retrieve all the sales orders and the items ordered in the system. Return just the sales order number, date, item name, rate and quantity, amount and the customer name:

                '''sql
                SELECT t.tranId AS "Sales Order Number", 
                    t.trandate AS "Sales Order Date", 
                    e.entityid AS "Customer Name", 
                    itm.itemid AS "Item Name", 
                    tl.quantity AS "Quantity", 
                    tl.rate AS "Rate", 
                    tl.amount AS "Item Amount"
                FROM entity e, transaction t, transactionLine tl, item itm
                WHERE t.type = 'SalesOrd'
                AND e.id = t.entity
                AND t.id = tl.id
                AND tl.item = itm.id
                '''
  ### Example Query: Retrieve a count of all the open invoices associated with Customer number 'Cust1853':
        '''sql
        SELECT COUNT(t.id) AS open_invoices
        FROM transaction t, entity e , transactionstatus ts
        WHERE  t.entity = e.id 
        and t.status = ts.id 
        and e.entityid = 'Cust1853'
          AND t.type = 'CustInvc'
        AND t.type = ts.trantype
        AND ts.name = 'Open'
        '''
### Example Query: Retrieve a count of all the purchase orders that are  'Pending Receipt'
"sql
SELECT t.tranid AS "Purchase Order Number", t.trandate AS "Order Date", e.entityid AS "Vendor Name", t.foreigntotal AS "Total Amount", t.status AS "Status" FROM transaction t INNER JOIN entity e ON t.entity = e.id INNER JOIN transactionstatus ts ON t.status = ts.id WHERE t.type = 'PurchOrd' AND ts.name = 'Pending Receipt' and ts.trantype = 'PurchOrd'
"

### Example Query
retrieve a count of Journals pending approval
SELECT COUNT(t.id) AS pending_journals  FROM transaction t, transactionstatus ts WHERE type = 'Journal' and ts.id = t.status and ts.trantype = t.type and ts.name = 'Pending Approval'
